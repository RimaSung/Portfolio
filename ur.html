UrCosme Website

Project type:
RWD Website
Duration:
2017-2018
Role:
UI / UX Designer
Methods:
UX Interview / User Journey / User Story / User Flow / Lo-fi Prototyping

OVERVIEW

UrCosme(@cosme) is the indicator cosmetics social community that allow Beauty lovers share their reviews on over 100k+ product pages. Those data provide a reliable reference value when consumers purchase products on the internet, brick-and-mortar stores, or @cosme physical store. This project demonstrates how to optimize the browsing process and drive users to be customers.

ACHIEVEMENTS

  • Systematized design patterns and styles for a responsive website

  • Draw digital users from online channels to make in-store purchases by conducting field studies to refine customer journeys, leading to a 60% purchase intention increase

  • Redesigned user interface and brand voice to encourage users to generate content on a social media platform for beauty products boosted web traffic by 42% and 2000+ new user creations in 6 months
