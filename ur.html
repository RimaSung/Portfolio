UrCosme Website

Project type:
RWD Website
Duration:
2017-2018
Role:
UI / UX Designer
Methods:
UX Interview / User Journey / User Story / User Flow / Lo-fi Prototyping

OVERVIEW

UrCosme(@cosme) is the indicator cosmetics social community that allow Beauty lovers share their reviews on over 100k+ product pages. Those data provide a reliable reference value when consumers purchase products on the internet, brick-and-mortar stores, or @cosme physical store. This project demonstrates how to optimize the browsing process and drive users to be customers.

ACHIEVEMENTS

  • Systematized design patterns and styles for responsive websites

  • Guided the product roadmap, including website and physical stores, by utilizing service design and UX insights

  • Integrated testing results with Google Analytics and heat map to design campaign placement, and embedded messages in over 100+ hits in all sections. Grew web traffic by 42% in 6 months
